Pivot Table Basics

Analize and Summarize
You can use pivot tables to analyze any set of data, quickly group and summarize values, and identify Interesting patterns and trends.

Fields and Source Data
Pivot tables are based on fields which correspond to columns in a separate table of source data. You choose which fields to include in the pivot table. In this example, the Sales Pivot table uses values from the Sales table.

Columns, Rows, and Values
Use fields to create columns and rows in the pivot table and to specify which data is summarized. In this example, the month columns are based on the Date field and subtotals are calculated using data from the Units field.

Pivot Table Options
To edit a pivot table, select it, click or tap the Organize button in the toolbar, and then choose Pivot Options. You can add, remove, and reorder fields. and choose options for how data is summarized, sorted, labeled, and more.
